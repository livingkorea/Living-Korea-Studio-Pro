# Living Korea Studio Pro

한국어 음성 입력, 영어 번역, 유튜브 대본 메모장 프로그램입니다.  
Korean voice input, English translation, and YouTube script notes.

## v0.3 변경사항 / Changes
- 한국어 + 영어 이중 언어 UI 적용
- 버튼, 상태 메시지, 안내창 모두 한국어/영어 병기
- 마이크 확인 버튼 유지
- GitHub Actions 빌드 경로 확인 완료

## 사용 전 확인 / Before Use
마이크 기능은 Windows에 한국어 음성 인식 엔진(ko-KR)이 설치되어 있어야 정상 작동합니다.

프로그램 실행 → `마이크 확인 / Check Mic` 버튼 클릭

ko-KR이 없으면:
Windows 설정 → 시간 및 언어 → 언어 및 지역 → 한국어 → 언어 옵션 → 음성 인식 설치
