# LivingKoreaStudio Whisper v1.2.4 Local Verified

이 버전은 GitHub Actions 산출물 이름을 `EXERUN`으로 고정한 버전입니다.

## 사용 방법

1. 이 ZIP을 풀어서 GitHub 저장소에 업로드합니다.
2. Actions 탭에서 `Build EXERUN` workflow를 실행합니다.
3. 성공 후 Artifacts에서 `EXERUN`을 다운로드합니다.
4. `EXERUN.zip`을 풀고 `LivingKoreaStudio.exe`를 실행합니다.

## 핵심 변경

- publish 출력 폴더를 `EXERUN`으로 고정
- Artifact 이름을 `EXERUN`으로 고정
- `EXERUN.zip` 자동 생성
- `EXERUN-folder`도 추가 업로드
- Whisper Runtime DLL 포함 여부 검증 유지
